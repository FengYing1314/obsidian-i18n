Obsidian i18n is a desktop plugin for [Obsidian](https://obsidian.md). It extracts translatable text from plugins and themes, then manages that text through AST editing, AI translation, apply/restore flows, Cloud downloads, and GitHub publishing.

<Note>
  **Terminology**

  - **Obsidian i18n** is the product name.
  - **I18N** is the actual display name in Obsidian's installed plugin list.
  - Locale codes such as `zh-cn`, `ja`, and `fr` belong to plugin translations.
  - The Mintlify docs site uses `zh-Hans` and `en` separately.
</Note>

## Workflow overview

<Steps>
  <Step title="Install the plugin">
    Add Obsidian i18n to the current vault through BRAT or the release ZIP.
  </Step>
  <Step title="Connect an AI provider">
    Configure a provider profile in **Settings → Community Plugins → I18N → `语言模型`**.
  </Step>
  <Step title="Extract and translate">
    Open the Manager, use the `插件` or `主题` tabs, extract strings, then translate them with AI or by hand.
  </Step>
  <Step title="Apply, restore, and share">
    Use **Apply** and **Restore** from the Manager; export local archives from `管理`, or publish to GitHub from the Cloud view.
  </Step>
</Steps>
