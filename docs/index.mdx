Obsidian i18n 是一款用于 [Obsidian](https://obsidian.md) 的桌面插件。它把插件与主题里的可翻译文本提取出来，再通过 AST 编辑器、AI 翻译、应用与还原、Cloud 下载与 GitHub 发布等流程，把译文管理成独立的本地译文源。

<Note>
  **术语说明**

  - **Obsidian i18n** 是产品名称。
  - **I18N** 是 Obsidian 已安装插件列表中的实际显示名称。
  - `zh-cn`、`ja`、`fr` 这类值是插件译文语言码。
  - Mintlify 文档站语言由 `zh-Hans` 和 `en` 控制，和插件译文语言码不是同一套配置。
</Note>

## 工作流概览

<Steps>
  <Step title="安装插件">
    通过 BRAT 或发行版 ZIP，把 Obsidian i18n 安装到当前 vault。
  </Step>
  <Step title="连接 AI 服务">
    在 **设置 → 社区插件 → I18N → 语言模型** 中配置 provider profile。
  </Step>
  <Step title="提取并翻译">
    打开管理中心，在 `插件` 或 `主题` 标签中提取字符串，再执行 AI 或手动翻译。
  </Step>
  <Step title="应用、还原与共享">
    在管理中心执行 **应用 / Apply** 与 **还原 / Restore**；在 `管理` 标签导出本地归档，或在 Cloud 视图发布到 GitHub 仓库。
  </Step>
</Steps>
