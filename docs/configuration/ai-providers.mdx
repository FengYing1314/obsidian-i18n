## 添加一个 provider profile

<Steps>
  <Step title="打开语言模型标签页">
    进入 **设置 → 社区插件 → I18N → 语言模型**。
  </Step>
  <Step title="选择服务商">
    在 provider 下拉框中选择目标平台。
  </Step>
  <Step title="创建或切换 profile">
    当前 UI 支持新增、切换和删除 profile。
  </Step>
  <Step title="填写连接信息">
    新建 profile 时，插件会写入该服务商的默认 URL 和默认模型；你再补充 API Key、模型名和价格字段。
  </Step>
  <Step title="执行测试连接">
    点击测试按钮，运行连接诊断。
  </Step>
</Steps>

## 当前页中的关键字段

<AccordionGroup>
  <Accordion title="URL / API Key / Model">
    这些字段属于当前 provider profile，彼此独立保存。
  </Accordion>

  <Accordion title="priceInput / priceOutput">
    这两个字段驱动费用预估，单位是人民币 / 每百万 Token。
  </Accordion>

  <Accordion title="llmResponseFormat">
    这个全局字段在 `语言模型` 页面编辑，当前默认值是 `text`。
  </Accordion>
</AccordionGroup>

## 不在当前页直接编辑的字段

下面这些全局翻译参数由 AST / Regex / Theme 的 AI 面板读写：

- `llmLanguage`
- `llmStyle`
- `llmBatchSize`
- `llmConcurrencyLimit`
- `llmTimeout`

<Note>
  Ollama 默认地址是 `http://localhost:11434`。
</Note>
